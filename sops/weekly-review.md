# SOP: Weekly Review

## Purpose
End-of-week review to track progress on SSPs, identify trends across customers, and plan the upcoming week. This feeds into your operational excellence metrics.

## When to Run
- **Frequency:** Every Friday afternoon (or last working day of the week)
- **Estimated Time:** 5 minutes (automated), 20-30 minutes (review + planning)

## Prerequisites
- Daily kickoff SOPs have been running throughout the week
- Customer folders populated with context

## Steps

### Step 1: Week in Review
```
Summarize my week:
- How many customer interactions did I have?
- What tickets were opened, updated, and closed?
- Any escalations or critical events?
- What action items did I complete vs. what's still open?
```

### Step 2: SSP Progress Check
```
For each of my customers, check:
- SSP milestone progress (what's on track, what's behind)
- Any SSP items due in the next 2 weeks
- Recommendations for SSP updates based on this week's activity
```

### Step 3: Customer Health Scorecard
```
Create a health scorecard for each customer:
| Customer | Open Cases | Escalations | SSP Progress | Health Events | Risk Level |
```

### Step 4: Trend Analysis
```
Compare this week to last week:
- Are case volumes trending up or down?
- Any recurring issues across customers?
- Cost anomalies or optimization opportunities?
```

### Step 5: Next Week Planning
```
Based on the review:
1. What are my top 3 priorities for next week?
2. Which customers need proactive outreach?
3. Any prep needed for upcoming cadence calls?
4. Time blocks to schedule for deep work (SSP building, certifications, initiatives)
```

## Expected Output
Weekly review document saved to `customers/_weekly-reviews/YYYY-WXX.md` containing:
- Week summary metrics
- Customer health scorecard
- SSP progress tracker
- Next week's plan with time blocks

## Customization
- Add team-level metrics if you're tracking across multiple TAMs
- Include certification study time in the planning section
- Add initiative progress tracking (operational excellence playbook, presentations)
