## Working Agreements

1. **SOPs are tool-agnostic** — built in markdown/YAML, work across Kiro, Quick, UNO, and CLI
2. **Don't force one tool** — each person uses what works best for them
3. **Share and reuse** — all SOPs go to GitLab (Olu's repo) AND GitHub (Amar's repo)
4. **Weekly sync** — Tuesdays 10:30 AM CEST
5. **Slack channel** — #project-ulduz for async updates
6. **Asana** — Amar tracks overall project progress in "Amar / Nick 1:1" board
